### Install

```sh
yarn
```

```sh
yarn build
```

Development in browser: (no interaction with real elastic server)

```sh
yarn dev
```

Development preview as an app:

```sh
yarn start
```

App build:

```sh
yarn dist-mac
yarn dist-win
yarn dist-linux
yarn dist #for all
```
