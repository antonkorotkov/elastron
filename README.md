# Elastron

### Install

```sh
yarn
```

Development in browser: 

```sh
yarn dev
```

Development preview as an app:

```sh
yarn start
```

App build:

```sh
yarn dist-mac #for mac
yarn dist-win #for windows
yarn dist     #for both
```

App relase

```sh
GH_TOKEN=<github token> yarn release-mac
GH_TOKEN=<github token> yarn release-win
```
